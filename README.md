# Muslim Life 360 Website

Static public website and social media launch pack for Muslim Life 360.

Open `index.html` directly in a browser. No package installation or development server is required.

## Before public launch

1. Replace the availability email button with live App Store and Google Play links.
2. Confirm permission to use every distributed audio and image asset.
3. Review the privacy policy against the final app behavior and cloud database rules.
4. Add the final hosted website URL to App Store Connect and Google Play Console.
5. Create social accounts using `SOCIAL_MEDIA_KIT.md` and keep one consistent handle.

## Files

- `index.html`: main product website
- `styles.css`: responsive visual design
- `script.js`: mobile navigation and current year
- `privacy.html`: public privacy policy
- `terms.html`: public terms of use
- `SOCIAL_MEDIA_KIT.md`: bios, posts and setup checklist
